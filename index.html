Инструкция по использованию

1. Для начала работы зарегистрируйтесь или войдите в систему.

2. После входа вы попадёте в панель управления учителя.

3. Выберите класс для работы с учениками.

4. В таблице успеваемости вы можете:

  • Добавлять/удалять учеников
  • Редактировать оценки по предметам
  • Добавлять/удалять предметы
  • Просматривать средний балл и статус ученика

5. Все изменения сохраняются автоматически.

6. Для выхода из системы нажмите кнопку "Выйти".
